About Us


We are Team Afternoon-4 and our project is about bringing easy-to-read Olympics statistics to you, the user! This site is split into three main groups, Countries, Host Cities, and Sports.
Countries contains a description of each participating country.
Host Cities brings you to a list of all of the Olympic games by year and general stats of each game.
Sports describes all of the olympic events that have been held since the inception of the Games.


Our Purpose

Bringing Olympic info to help users feel more connected to their hometown.

Our Audience

Anyone who wants to learn more about the Games are welcome here!

Tools

Visual Studio Code

Our main editor. We compiled, debugged, and edited our code. This also allowed us to easily see our files’ hierarchy, where they’re located, and has the accessibility to use the command line and instantly load the website into our local machine.

Google Cloud Platform

This website is hosted on Google Cloud Platform. We are using the Python 3 standard environment and Flask to serve requests

Scrapy

Our implementation to interact with our RESTful API. Scrapes the data off the website to make it our own. This allows us to create json files which we can then parse, filter, sort, and use. We also used python's requests library and beautifulsoup4 to scrape websites that did not have a REST API already. Requests obtains request data from the servers and bs4 is used to parse the data.
